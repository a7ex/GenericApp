  • nearestSuperview(ofType:)

    Get the nearest parent of a given type

    Go recursively through the parent views and return the first one, which matches the provided type

    E.g. in order to find the tableView which contains a certain cell: let enclosingTableView = tableCell.nearestSuperview(ofType: UITableView.self)
