Optional

enum Optional<Wrapped> : ExpressibleByNilLiteral

Extend an Optional which wraps a String value type

  • hasNonEmptyStringContent

    Extend an Optional which wraps a String value type so that we can check in one go, whether it is nil OR isEmpty

    so instead of requiring something very common in our code like:

    if let unwrappedString = someOptionalString,
     !unwrappedString.isEmpty {
     ... do something if unwrapped 'unwrappedString' is not nil and not empty
}

    we can instead just use:

    if someOptionalString.hasNonEmptyStringContent {
     ... do something if unwrapped 'unwrappedString' is not nil and not empty
}

    Swift

  • nonEmptyString

    Extend an Optional which wraps a String value type so that we can assign only if the string is not nil and not empty

    so instead of requiring something very common in our code like:

    if let unwrappedString = someOptionalString,
     !unwrappedString.isEmpty {
     ... do something with non empty, unwrapped 'unwrappedString'
}

    we can instead just use:

    if let unwrappedString = someOptionalString.nonEmptyString {
     ... do something with non empty, unwrapped 'unwrappedString'
}

    Swift

  • nonEmptyString(trimmingChars:)

    Extend an Optional which wraps a String value type so that we can assign only if the string is not nil and not empty where ‘empty’ is defined here only after trimming the provided Characters This allows us to also treat strings like “ or\t\t \n” as empty

    Usage:

    if let visibleString = someOptionalString.nonEmptyString(trimmingChars: CharacterSet.whitespacesAndNewlines) {
     ... do something with non empty, visible, unwrapped 'unwrappedString'
}

  • printableNonEmptyString

    Extend an Optional which wraps a String value type so that we can assign only if the string is not nil and not empty where ‘empty’ is defined here only after trimming whiteSpaceAndNewline characters

    Usage:

    if let visibleString = someOptionalString.printableNonEmptyString {
     ... do something with non empty, visible, unwrapped 'unwrappedString'
