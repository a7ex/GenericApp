  • viewController(withIdentifier:)

    Instantiates a Viewcontroller from a given storyboard either by sceneId or taking the initial vc

    Declaration

    Swift

    func viewController(withIdentifier identifier: String?) -> UIViewController?

    Parameters

    identifier

    The sceneId of the ViewController Scene in the storyboard. If missing or empty instantiateInitialViewController() is called instead and returns the initial ViewController defined in the storyboard

    Return Value

    A ViewController (or its subclass) for the specified identifier.

  • init(name:)

    Creates and returns a storyboard object for the specified storyboard resource file in the main bundle of the current application.

    Declaration

    Swift

    convenience init(name: String)

    Parameters

    name

    The name of the storyboard resource file without the filename extension.

    Return Value

    A storyboard object for the specified file. If no storyboard resource file matching name exists, an exception is thrown.
